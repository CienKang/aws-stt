# HOW TO RUN LOCALLY


### Create an .env file with format
```
OPENAI_BASE_URL=https://openai.prod.ai-..............

OPENAI_API_KEY=eyJ..........

AWS_ACCESS_KEY_ID=A........

AWS_SECRET_ACCESS_KEY=D........

AWS_SESSION_TOKEN=F..........

S3_BUCKET=...........
```


### just compose up 
```
docker compose up --build
```
